# Tecnologias Utilizadas


Análisis de factores asociados a la depresión y la salud mental



### Lenguajes de Programacion
- Scala

### Framework
- Apache Spark (con uso de Spark SQL para consultas estructuradas).

### Entorno de Trabajo
- Databricks: Plataforma basada en la nube para el análisis de datos grandes. 
Integra Apache Spark y es ideal para equipos colaborativos.

### Formato de Datos
- CSV
